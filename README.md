# Finto 📱💸

[cite_start]**Finto** — це мобільний застосунок, призначений для ведення особистого фінансового обліку, що поєднує класичне ручне введення даних із технологією оптичного розпізнавання символів (OCR)[cite: 198, 199].

## 🎯 Мета проєкту

[cite_start]Створити зручний, швидкий та приватний інструмент для контролю витрат, який мінімізує час на ведення бухгалтерії завдяки автоматизації та інтуїтивній візуалізації фінансового стану[cite: 203].

---

## 🚀 Основні можливості

### 📸 Гібридне введення даних
[cite_start]Ключова особливість системи — гнучкість введення[cite: 200]:
* [cite_start]**Ручне введення:** Швидке додавання суми та вибір категорії для тих, хто звик контролювати все самостійно[cite: 200, 215].
* [cite_start]**OCR Сканування:** Автоматичне зчитування даних з чеків за допомогою камери смартфона[cite: 200].

### 🛡️ Приватність та оптимізація (OCR Module)
Система працює за принципом **"Process & Discard"**:
* [cite_start]Запуск камери безпосередньо з інтерфейсу[cite: 219].
* [cite_start]Автоматичне розпізнавання тексту та категоризація даних[cite: 220, 221].
* **Важливо:** Фотографія чека видаляється одразу після обробки. [cite_start]Вона не зберігається в пам'яті телефону та не завантажується у хмару, що гарантує приватність та економить місце[cite: 201, 209, 222].

### 📊 Аналітика та візуалізація
[cite_start]Застосунок структурує хаотичні витрати у зрозумілу інфографіку[cite: 207]:
* [cite_start]**Pie Chart:** Діаграма розподілу витрат за категоріями[cite: 225].
* [cite_start]**Bar Chart:** Графік динаміки витрат за тиждень або місяць[cite: 226].

### ⚙️ Управління
* [cite_start]Редагування та видалення існуючих записів[cite: 216].
* [cite_start]Додавання коментарів до транзакцій[cite: 217].
* [cite_start]Створення власних категорій витрат[cite: 229].

## 💡 Які проблеми вирішує Finto?

1.  [cite_start]**Складність обліку:** Вирішує проблему рутини, зводячи процес внесення даних до одного натискання кнопки сканування[cite: 205, 206].
2.  [cite_start]**Інформаційна "сліпота":** Надає наочні графіки для розуміння того, куди зникають кошти[cite: 207].
3.  [cite_start]**Приватність:** На відміну від аналогів, Finto не зберігає фото чеків, обробляючи дані "на льоту"[cite: 209, 210].
4.  [cite_start]**Людський фактор:** Автоматизація знижує ризик помилок, які виникають при ручному переписуванні сум з довгих чеків[cite: 211].
