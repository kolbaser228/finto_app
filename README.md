# Finto 📱💸

**Finto** — це мобільний застосунок, призначений для ведення особистого фінансового обліку, що поєднує класичне ручне введення даних із технологією оптичного розпізнавання символів (OCR).

## 🎯 Мета проєкту

Створити зручний, швидкий та приватний інструмент для контролю витрат, який мінімізує час на ведення бухгалтерії завдяки автоматизації та інтуїтивній візуалізації фінансового стану.

---

## 🚀 Основні можливості

### 📸 Гібридне введення даних
Ключова особливість системи — гнучкість введення:
* **Ручне введення:** Швидке додавання суми та вибір категорії для тих, хто звик контролювати все самостійно.
* **OCR Сканування:** Автоматичне зчитування даних з чеків за допомогою камери смартфона.

### 🛡️ Приватність та оптимізація (OCR Module)
Система працює за принципом **"Process & Discard"**:
* Запуск камери безпосередньо з інтерфейсу.
* Автоматичне розпізнавання тексту та категоризація даних.
* **Важливо:** Фотографія чека видаляється одразу після обробки. Вона не зберігається в пам'яті телефону та не завантажується у хмару, що гарантує приватність та економить місце.

### 📊 Аналітика та візуалізація
Застосунок структурує хаотичні витрати у зрозумілу інфографіку:
* **Pie Chart:** Діаграма розподілу витрат за категоріями.
* **Bar Chart:** Графік динаміки витрат за тиждень або місяць.

### ⚙️ Управління
* Редагування та видалення існуючих записів.
* Створення власних категорій витрат.

## 💡 Які проблеми вирішує Finto?

1.  **Складність обліку:** Вирішує проблему рутини, зводячи процес внесення даних до одного натискання кнопки сканування.
2.  **Інформаційна "сліпота":** Надає наочні графіки для розуміння того, куди зникають кошти.
3.  **Приватність:** На відміну від аналогів, Finto не зберігає фото чеків, обробляючи дані "на льоту".
4.  **Людський фактор:** Автоматизація знижує ризик помилок, які виникають при ручному переписуванні сум з довгих чеків.

## 🛠️ Технічний стек (Tech Stack)

Проєкт розробляється з використанням наступних технологій та бібліотек:

* **Мова програмування:** Java
* **Платформа:** Android SDK
* **Архітектура:** MVVM (Model-View-ViewModel)
* **База даних:** **Room** (SQLite).
* **OCR (Розпізнавання тексту):** **Google ML Kit (Text Recognition)** — для швидкої обробки чеків "на льоту" без збереження фото.
* **Графіки:** **MPAndroidChart** — для побудови діаграм (Pie Chart, Bar Chart) та візуалізації аналітики.
* **UI:** XML / Material Design Components.
